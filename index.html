React Bootstrap Table

react-bootstrap-table is a React component for Bootstrap table. It's a configurable, functional table component.
It make you build a Bootstrap Table more efficiency and easy in your React application.


Functional Table

react-bootstrap-table built with Bootstrap and support a lots of feature on table component, include column align, hidden, sorting, formating, filter, styling, pagination and search.

Manipulation

Sometimes, a basic CRUD on table data is very important for table component. So react-bootstrap-table enabled the row selection and cell editing on table. In addition, row insert and remove also work on table.

Advance features

We provide some advance function like update table on fly, dynamic row selection and event hook for some operation.

A Quick Demo


function format(cell, row){
  return '<i class="glyphicon glyphicon-usd"></i> ' + cell;
}
//Row select setting
var selectRowProp = {
  mode: "checkbox",  //checkbox for multi select, radio for single select.
  clickToSelect: true,   //click row will trigger a selection on that row.
  bgColor: "rgb(238, 193, 213)"   //selected row background color
};

<BootstrapTable data={productLong} striped={true} hover={true} condensed={true} pagination={true}
              selectRow={selectRowProp} insertRow={true} deleteRow={true} columnFilter={true} search={true}>

  <TableHeaderColumn dataField="id" isKey={true} dataAlign="right" dataSort={true}>Product ID</TableHeaderColumn>
  <TableHeaderColumn dataField="name" dataSort={true}>Product Name</TableHeaderColumn>
  <TableHeaderColumn dataField="price" dataAlign="center" dataFormat={format}>Product Price</TableHeaderColumn>
</BootstrapTable>
